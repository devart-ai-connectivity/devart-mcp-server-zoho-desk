# MCP Server Usage Guide

## Description

This MCP server enables you to explore and interact with a relational database using a Devart .NET connector.

You have access to:

* Tables — database objects that store structured data and can be queried
* Columns — fields that define the structure and data types within a table
* Indexes — database objects that optimize data access and improve query performance
* Foreign keys — constraints that establish and enforce relationships between tables
* Procedures and functions — callable database routines used to encapsulate logic and operations
* Query execution — the ability to run SQL statements to read and modify data

With the provided tools, you can:

* Retrieve metadata about the database schema, including tables, columns, indexes, and foreign keys
* Execute SQL SELECT statements to read data
* Execute DML statements (INSERT, UPDATE, DELETE, and others) to modify data
* Enumerate stored procedures and functions and inspect their parameters
* Invoke procedures and functions and access their results
